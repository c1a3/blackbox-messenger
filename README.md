### Setup .env file

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Build the app
`/root directory`
```shell
npm run build
```

### Initialize the app

```shell
cd frontend
npm run dev
```
```shell
cd backend
npm start
```
